# 29524233_TFM
Trabajo de Fin de Máster. Herramientas en MATLAB para el dimensionado óptimo de sistemas aislados híbridos mediante GA, ACOR y CMA-ES, con evaluación de fiabilidad por simulación de Monte Carlo, y para el estudio de cooperación energética entre viviendas de una microrred. Incluye interfaces gráficas.
